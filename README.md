# clear_plan

## 功能描述
这是一个个人小型项目计划清，在首页为用户倒计时提示最紧急最重要的事情，如距离高考仅剩72天10小时6分钟3秒。
功能页包括（1）紧急且重要、紧急但不重要、重要但不紧急、不重要不紧急的计划的管理。（2）提供计划完成时的记录，如结束一项会议时的会议记录等。
